## 1. システム概要

個人利用のゴルフラウンド記録・距離測定Webアプリ。
Google Maps（衛星写真）上でショットを記録し、クラブ・飛距離・スコアを管理する。

### 技術スタック

| 項目 | 内容 |
|---|---|
| ホスティング | GitHub Pages（無料） |
| 地図 | Google Maps JavaScript API（hybridモード） |
| データ保存 | localStorage（最大30ラウンド） |
| フレームワーク | なし（Vanilla HTML/CSS/JS） |
| フォント | Noto Sans JP / Bebas Neue（Google Fonts） |

## 4. 実装済み機能

### 📏 測定モード

- 地図タップ → 起点（ティーまたは前打地点）からの距離（青線）を表示
- 同時に最近ピンまでの残り距離（黄線）を表示
- F・C・Bのマーカーをタップするとそのピンへの距離に切替
- 地図上に距離ラベル（青：飛距離、黄：残り距離）をリアルタイム表示
- 右上パネルに測定距離を追加表示

### 🏌️ 記録モード（タブ式パネル）

- 地図タップ → ボール落下地点として登録
- **🏌️ 記録タブ（デフォルト）**：クラブ選択（12種類・6列2行）
  - `1W / 3W / 4H / 5I / 6I / 7I / 8I / 9I / PW / AW / SW / PT`
- **📏 飛距離確認タブ**：carry（飛距離）・remaining（残り距離）を表示
- ⛳ IN ボタンでカップイン（打数+1で正確にカウント）
- スコア手動変更可能（イーグル〜トリプルボギー）

### 📊 カップイン後サマリー

- ショット詳細：1打目から全打のクラブ・飛距離・残り距離
- スコアカード：全ホール分を表形式で表示（色付きスコアバッジ）
- 「次のホールへ」ボタンで手動遷移

### 🏅 スコア表示

- ホールストリップに色付きスコアバッジ
  - イーグル（ピンク）/ バーディ（青）/ パー（緑）/ ボギー（オレンジ）/ ダブル（赤）/ トリプル（濃赤）
- 履歴パネルに9ホール分のスコアグリッド＋合計スコア

### 💾 データ保存

- localStorageに自動保存（最大30ラウンド）
- 保存データ：日付・ゴルフ場名・コース名・全ショット情報（クラブ・飛距離・座標）
- 📋 履歴ボタンから過去ラウンドを参照可能

### 📍 GPS

- 現在地リアルタイム追尾
- 記録モード中にGPSボタン → 現在地でショット登録

### 🗺 地図操作

- `⛳↑` ボタン：ティー→グリーン方向が上になるよう回転
- `🧭N` ボタン（回転中）：北向きに戻す
- `📐` ボタン：ヤード情報パネルの開閉

## 6. picker.html（座標登録ツール）

**URL：** https://keeeeei-soeda.github.io/golf_shot_map/picker.html

```
ゴルフ場名・都道府県・GCID・コース名・CID を入力
  ↓
衛星写真地図上でホール選択 → 4クリック（T/F/C/B）
  ↓
自動で次の地点に切替（全4点完了後は手動で次ホールへ）
  ↓
「courses.js形式でコピー」ボタン
  ↓
courses.js に貼り付けて完了
```

- 1ホール = 4クリック、9ホール = 36クリック（約10〜15分）
- 完了ホールに緑ドット表示
- parボタンで各ホールのPAR設定
